* 👑 יוסף נשאר יוסף גם בכלא

* 🧠 פתרון חלומות פרעה

* 🏛️ יוסף משפיע על מצרים כולה
* 🌍 אור פרטי הופך אור כללי

* 💰 שפע = ניסיון
* ⚖️ איזון, ענווה, אחריות
